# ☕ Aja Cafe - Smart Inventory System

A modern, serverless Progressive Web App (PWA) built specifically for fast-paced cafe environments. Designed for mobile-first usability with a focus on quick, crash-proof inventory logging.

## ✨ Key Features
* **Real-Time Sync:** Powered by Firebase Firestore for instant stock updates across all devices.
* **Smart UI/UX:** Features "Stock Pulse" low-inventory warnings, haptic feedback, and thumb-friendly bottom sheets.
* **Role-Based Access:** Secure login system separating Admin controls from standard worker access.
* **Offline-Ready:** Built as a PWA with IndexedDB persistence to survive spotty cafe internet connections.
* **Action Logging:** Complete historical ledger of every stock addition and deduction.

## 🛠 Tech Stack
* HTML5 / CSS3 / Vanilla JavaScript
* Google Firebase (Authentication & Cloud Firestore)
* Hosted on Netlify
